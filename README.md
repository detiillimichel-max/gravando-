# Gravador de Voz

Gravador PWA simples, offline, com visual escuro igual ao app nativo do Android.

## O que faz
- Grava áudio pelo microfone (MediaRecorder API)
- Mostra waveform animado enquanto grava
- Salva com nome, data, duração e tamanho
- Lista por mês em "Todas as gravações"
- Reproduz, renomeia, exclui e baixa em .webm
- Funciona 100% offline (salva no navegador)

## Como usar
1. Abra o index.html
2. Toque no botão vermelho para gravar
3. Toque no quadrado para parar
4. Adicione um nome > Salvar
5. A gravação aparece na lista

## Instalar como app
1. Publique no GitHub Pages
2. No Chrome > ⋮ > Instalar aplicativo
3. Permita o microfone na primeira vez

Privacidade: tudo fica só no seu celular, nada vai para servidor.
